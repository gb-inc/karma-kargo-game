# Karma Kargo

A turn-based trading game where you captain a cargo ship between islands, buying and selling goods to build your fortune. Navigate dynamic markets, manage provisions, and compete for the highest net worth across 50 turns—then keep sailing in sandbox mode.

This is just the beginning—an early proof-of-concept—MUCH more to come...

**Website:** [karmakargo.com](https://www.karmakargo.com)

## Download

Head to the [Releases](https://github.com/gb-inc/karma-kargo-game/releases) page and download the latest `KarmaKargo.exe`. No installation required — just run the EXE.

**Requirements:** Windows 10 or later (x86)
